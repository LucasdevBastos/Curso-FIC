Google Docs

Como formatar textos de forma simples:

Vamos aprender o básico para deixar seus documentos com uma aparência profissional. Não se preocupe, é mais fácil do que parece!

  • Deixar em negrito: Selecione o texto e clique no ícone "B".
  • Mudar a cor: Selecione o texto e clique no ícone "A" com uma barrinha colorida embaixo.
  • Adicionar uma imagem: No menu superior, clique em `Inserir > Imagem`.

Google Planilhas

Como fazer uma soma automática:

Planilhas são ótimas para organizar números. Veja como é simples fazer o computador trabalhar para você e somar valores automaticamente.

  • Coloque um número na célula A1 (ex: 10).
  • Coloque outro número na célula A2 (ex: 20).
  • Na célula A3, digite =SOMA(A1:A2) e aperte Enter. A mágica acontece!

Canva

Como criar uma arte bonita rapidamente:

O Canva é uma ferramenta fantástica para criar designs mesmo sem ser um designer. O segredo é usar os modelos prontos!

  • Na tela inicial, procure por um modelo (ex: "convite de aniversário").
  • Clique no modelo que gostar para abri-lo.
  • Clique nos textos e imagens para editar com suas informações. É simples assim!
