O Projeto de Inclusão Digital

Assista ao vídeo para saber mais sobre o projeto!

Esta foi a primeira edição de um sonho: um projeto de inclusão digital pensado para a nossa comunidade. O objetivo foi oferecer, de forma gratuita e acolhedora, os conhecimentos essenciais de informática para todos que desejavam aprender e se conectar com o mundo digital.

Acreditamos que o conhecimento em tecnologia abre portas, cria oportunidades e dá mais autonomia para as pessoas no dia a dia. Este site é uma celebração do esforço de todos os envolvidos e um material de apoio para que o aprendizado continue!

O que Nós Ensinamos

Hardware

São as partes físicas do computador, tudo aquilo que podemos tocar: processador, memória RAM, disco rígido (HD/SSD), monitor, teclado e mouse.

Software

São os programas e instruções que fazem o hardware funcionar. Inclui os Sistemas Operacionais e os aplicativos que usamos.

Internet e Navegação

É a rede mundial que conecta computadores. Usamos navegadores para acessar sites e buscadores para encontrar informações.

Google Docs

O que é e para que serve?

O Google Docs é um editor de textos 100% online, perfeito para criar currículos, trabalhos e relatórios. Tudo fica salvo automaticamente na nuvem!

Dicas Essenciais

  • Formatação de Texto: Use a barra de ferramentas para alterar a fonte, o tamanho e a cor.
  • Listas e Alinhamento: Organize suas ideias com listas e use os botões de alinhamento.
  • Inserir Imagens e Links: Vá ao menu "Inserir" para adicionar imagens ou links.

Google Planilhas

O que é e para que serve?

É uma planilha digital ideal para organizar informações, criar listas, controlar finanças e fazer cálculos automáticos com fórmulas.

Primeiros Passos

  • Entendendo as Células: Cada retângulo é uma célula (ex: A1, B2).
  • Fórmulas Básicas: Comece com o sinal de igual (=). Para somar, use: =SOMA(A1:A2).
  • Formatando Células: Mude a cor de fundo ou formate números como moeda (R$).

Canva

O que é e para que serve?

O Canva é uma ferramenta de design gratuita para criar artes incríveis como posts para redes sociais, convites e apresentações, mesmo sem ser designer.

Como Criar sua Primeira Arte

  • Use os Modelos: Pesquise o que você quer criar e escolha um design pronto para editar.
  • Arraste e Solte: Personalize clicando nos elementos, trocando textos e arrastando suas imagens.
  • Baixe sua Arte: Clique em "Compartilhar" e "Baixar" para salvar seu design como PNG ou PDF.

Gov.br

O que é e para que serve?

O Gov.br é a plataforma do Governo Federal que unifica o acesso a milhares de serviços públicos digitais. Com uma única conta, você pode acessar serviços do INSS, Carteira de Trabalho Digital e muito mais.Pontos Importantes

  • Cadastro Único: Crie sua conta com seu CPF para ter acesso a tudo.
  • Níveis de Segurança: Sua conta pode ser Bronze, Prata ou Ouro.
  • App Gov.br: Baixe o aplicativo para facilitar o acesso e aumentar a segurança da sua conta.
